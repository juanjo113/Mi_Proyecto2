## Pruebas: Calificaciones y CE

* **CE**: Criterio de Evaluación.
* **PM**: Puntuacion Máxima.
* **Ps10**: Puntuación sobre 10.
* **RA**: Resultado de Aprendizaje.

Esto es un ejempo de dos tablas CE-PM en la programación Didáctica:
